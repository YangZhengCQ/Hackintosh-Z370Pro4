# Hackintosh-Z370Pro4
配置：OpenCore-0.5.7,MacOS-10.15.4,i9-9900k,RX580。  
  
·显卡必须使用免驱动的AMD显卡（如RX580@2304SP，Vega系列，RX5000系列）。  
  
·没有注入任何无线网卡与蓝牙，推荐使用免驱动的网卡。  
  
·没有技术支持，但是我（有空）会不断更新引导器版本和EFI以适配未来更高系统版本。
